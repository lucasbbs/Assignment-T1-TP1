Detailed Description

Padrão para representação de cliente.

Regras de formato:

  • Cidade é válida caso:
    • seja passado um valor dentro de "Hong Kong", "Bangkok", "Macau", "Singapura", "Londres", "Paris", "Dubai", "Delhi", "Istambul", "Kuala", "Lumpur", "Nova Iorque", "Antalya", "Mumbai", "Shenzen", "Phuket", ou;
    • não seja passado o objeto é instanciado com uma string vazia, "".
